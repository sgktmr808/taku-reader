# 卓リーダー

対談ログ（卓）をスマホで読むための、データを一切持たない静的ビューアー。

- **このリポジトリに卓の中身は含まれません。** 表示するデータは手元の `taku_bundle.json`
  （taku-bundle/1 形式）を画面から読み込みます。
- 読み込んだデータは**端末内（IndexedDB）にのみ**保存され、どこにも送信されません。
- PWA対応＝ホーム画面に追加すればオフラインで読めます。

## 使い方

1. 手元の書き出しスクリプトで `taku_bundle.json` を作る
2. AirDrop やクラウド経由でスマホに送る
3. 公開ページを開いて「バンドルを読み込む」→ 以後はオフラインで閲覧可
4. データを更新したい時は新しいバンドルを再読み込み（上書き）

## バンドル形式（taku-bundle/1）

```json
{
  "format": "taku-bundle/1",
  "exported_at": "2026-07-21T12:00:00",
  "count": 2,
  "talks": [
    {
      "id": "T0001",
      "file": "talk_example.md",
      "title": "題名",
      "genre": "ジャンル",
      "cast": ["話者A", "話者B"],
      "date": "2026-07-21",
      "body": "## 題名\n\n話者A：台詞…\n話者B：台詞…"
    }
  ]
}
```

`body` は行頭「名前：」の台詞行・`# 場面見出し`・地の文をそのまま持つMarkdown風テキスト。
`## 座組メモ` 節は折りたたみで表示され、`## 生成情報` 節は書き出し時に除外する想定。

## 開発

依存なしの単一 `index.html`。ローカル確認は `python3 -m http.server` などで。
`?src=<同一オリジンのJSON>` で起動時読み込み（開発・自己ホスト用）。
